## What is a Palindrome?

A palindrome is a word, number, or sequence that reads the same forward and backward.

## Definition

If the original value and its reverse are the same, then it is called a palindrome.

## Examples

Palindrome:
- 121 → Reverse is 121
- madam → Reverse is madam
- level → Reverse is level

Not a Palindrome:
- 123 → Reverse is 321
- hello → Reverse is olleh

## How to Identify a Palindrome?

1. Take the input value.
2. Reverse the value.
3. Compare the original value with the reversed value.
4. If both are equal → Palindrome.
5. If both are different → Not a Palindrome.
